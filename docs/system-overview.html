Settle Backend

Shared Expense Tracking System

Энэ систем нь групп дотор хамтын зардлыг бүртгэж, хэрэглэгч бүрийн авлага, өглөгийг group тус бүрээр тооцоолж, дараа нь бөөнд нь settlement хийх боломж олгоно.

Core Rule

Гол ойлголт

Нэг хэрэглэгч олон group-д байж болно. Balance нь зөвхөн group-ийн хүрээнд бодогдоно.

  • Group үүсгэсэн хүн автоматаар owner
  • Owner л member add/remove хийнэ
  • Member бүр expense бүртгэж болно
  • Balance нь expense + settlement дээрээс бодогдоно
Auth

Нэвтрэлт

Local JWT authentication ашиглаж байна.

  • POST /auth/sign-up
  • POST /auth/sign-in
  • GET /auth/me
Output

Системийн гаргалгаа

  • Group list
  • Expense history
  • Settlement history
  • Net balance
  • Pairwise payable / receivable
Main Flow

Үндсэн ажиллагааны урсгал

1
Хэрэглэгч бүртгүүлж нэвтэрнэ
JWT access token авч, хамгаалагдсан endpoint-ууд руу ашиглана.
2
Group үүсгэнэ
Group үүсгэсэн хэрэглэгч owner болно. Owner нь бусад хэрэглэгчдийг group руу нэмнэ.
3
Expense бүртгэнэ
Нэг payer нийт дүнг төлнө. Participant бүрийн share amount тусдаа хадгалагдана.
4
Balance бодно
Net balance болон pairwise өр авлагыг expense + settlement дээрээс aggregate хийнэ.
5
Settlement бүртгэнэ
Хоёр хэрэглэгчийн хооронд хийсэн бодит төлөлтийг бүртгэснээр тухайн pair-ийн өр буурна.
Balance Logic

Тооцоолол

Төлөв Утга
netBalance > 0 Авах ёстой
netBalance < 0 Өгөх ёстой
netBalance = 0 Тэнцсэн
Pairwise өр нь нэг хүний өрийг өөр хүнд шилжүүлэхгүй. Зөвхөн тухайн 2 хүний хоорондын expense ба settlement дээрээс цэвэр үлдэгдэл гаргана.

Expense Example

Expense хэрхэн өр үүсгэдэг вэ

Батаа нийт 36,000 төгрөг төлсөн гэж бодъё.

Хүн Өөрийн share Net effect
Батаа 12,000 +24,000
Ганаа 15,000 -15,000
Баяраа 9,000 -9,000
Pairwise Example

Settlement хэрхэн pairwise өрийг багасгадаг вэ

Хоёр хүний хоорондын бодит төлөлт тухайн pair-ийн өрөөс шууд хасагдана.

Алхам Үр дүн
Expense дараа Ганаа → Батаа = 15,000
Settlement 5,000 Ганаа → Батаа = 10,000
Бусад хүн Баяраагийн өр өөрчлөгдөхгүй
Frontend API

Web frontend-д хамгийн хэрэгтэй endpoint-ууд

Feature Endpoint Тайлбар
Login POST /auth/sign-in JWT token авна
Dashboard GET /users/me/balances Group бүр дээрх өөрийн товч balance
My activity GET /users/me/expenses Өөрийн expense history
My settlements GET /users/me/settlements Өөртэй холбоотой settlement history
Group detail GET /groups/:groupId Group мэдээлэл
Group members GET /groups/:groupId/members Гишүүдийн жагсаалт
Group expenses GET /groups/:groupId/expenses Expense жагсаалт
Group balances GET /groups/:groupId/balances Net balance + pairwise өр авлага
Create expense POST /groups/:groupId/expenses Шинэ зардал бүртгэх
Create settlement POST /groups/:groupId/settlements Төлөлт бүртгэх
